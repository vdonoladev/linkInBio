What is Zutrabio?

Zutrabio is a production-ready Link in Bio theme built with Astro 6. It comes with three visual themes, six ready-to-use persona profiles, 50+ platform integrations, full SEO, analytics support, and one-click deploy to Cloudflare Pages — all configurable from a single file.

3 Visual Themes
Minimal, Modern (glassmorphism), Neobrutalist — each with light & dark mode.
6 Profiles Included
Freelancer, Creator, Musician, Founder, Professional, Local Business.
50+ Platforms
Instagram, GitHub, Spotify, Gumroad, Calendly, and many more.
Full SEO Stack
Open Graph, Twitter Cards, Schema.org JSON-LD, sitemap, hreflang.
Cloudflare Ready
Deploy to Cloudflare Pages free tier in under 5 minutes.
One Config File
Everything editable from site-config.ts. No framework knowledge needed.

Quick Start

Get your page running locally in under 2 minutes.

1
Install dependencies
Requires Node.js 18+ and pnpm (or npm).
pnpm install
# or: npm install
2
Start the dev server
pnpm dev
# Opens http://localhost:4321
3
Edit your config
Open src/site-config.ts and fill in your identity, links, and content. The page hot-reloads on every save.
4
Build for production
pnpm build
# Output goes to dist/
5
Deploy
Push to GitHub and connect to Cloudflare Pages. See the Deploy section for step-by-step instructions.

Rule of thumb: 90% of customization happens in src/site-config.ts and your active profile file in src/profiles/. You rarely need to touch anything else.

site-config.ts

The single source of truth. Every section is documented inline — here's an annotated overview.

// ── 1. Active profile ──────────────────────────────────
export { pageItems } from "./profiles/freelancer";
// Options: freelancer | creator | musician | founder | professional | localbusiness

// ── 2. The full config ─────────────────────────────────
export const siteConfig: SiteConfig = {

  identity: { /* your name, avatar, banner, bio… */ },

  siteUrl: "https://yoursite.com",

  promo: {
    enabled: true,
    message: "✨ My new course is live!",
    cta: { label: "Check it out", url: "https://..." },
    dismissible: true,
  },

  theme: {
    default: "modern",     // "minimal" | "modern" | "neobrutalist"
    colorMode: "system",  // "light" | "dark" | "system"
    accentColors: { modern: "#6c47ff" },
    showThemeSwitcher: true,
  },

  analytics: { gtmId: "GTM-XXXXXXX", /* …other providers */ },
  seo:       { title: "Jane Doe — Designer", /* … */ },
  openGraph: { image: "/og/og-default.jpg", /* … */ },
  features:  { showPoweredBy: true, animateCards: true, /* … */ },
  footer:    { copyright: "", links: [] },
};

Identity

Controls the profile header — name, avatar, banner image, bio, location, and timezone.

FieldTypeDescription
type"person" | "organization"Affects Schema.org output

biostringShort bio line (plain text)
bioExtendedstringOptional second bio line

locationstringShown with a pin icon, e.g. "Santiago, Chile 🇨🇱"

Banner tip: Use a landscape image at least 1200×400px. Unsplash URLs work great. The image gets a subtle gradient overlay automatically.

Promo Banner

A dismissible announcement strip shown at the very top of the page. Visitors can close it — the choice is saved in localStorage.

promo: {
  enabled: true,     // set false to hide completely
  message: "✨ My new course is live!",
  cta: {
    label: "Enroll now",
    url:   "https://gumroad.com/...",
  },
  dismissible: true,  // show the × close button
},

The banner inherits the active theme's accent color as background — so it always looks on-brand without extra configuration.

Custom accent color

Override the accent color for any theme. Leave a key out to keep the built-in default.

accentColors: {
  minimal:      "#18181b", // default: near-black
  modern:       "#6c47ff", // default: vivid purple
  neobrutalist: "#ff6b35", // default: orange
},
Hex only. The foreground (white/black text on the accent) is computed automatically from the color's luminance — you don't need to set it.

Hiding the theme switcher

Once you've chosen your final look, hide the floating switcher so visitors can't change it:

showThemeSwitcher: false,

The switcher appears as a palette icon in the top-right corner on mobile and as a "Themes" pill in the bottom-left on desktop.

Duplicate any existing profile and edit it:

# 2. Edit the file with your content

# 3. Activate it in site-config.ts
export { pageItems } from "./profiles/myprofile";

Image Gallery

The gallery renders a responsive grid. Clicking any image opens a full-screen lightbox with navigation and touch swipe support.

{
  type: "rich-media",
  mediaType: "image-gallery",
  label: "Portfolio",
  images: [
    { src: "https://...", alt: "Project 1" },
    { src: "https://...", alt: "Project 2" },
    // up to any number of images
  ],
  description: "Selected work — 2022–2024",
  enabled: true,
}

Lightbox features: keyboard navigation (← → Esc), touch/pointer drag to swipe, image counter, smooth entrance animation. No external dependencies.

Deploy to Cloudflare Pages

The recommended platform — free tier, global CDN, security headers pre-configured.

1
Push your repo to GitHub
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/you/mysite.git
git push -u origin main
2
Create a Cloudflare Pages project
Go to dash.cloudflare.com → Pages → Create project → Connect to Git → Select your repo.
3
Set build settings
Build commandpnpm build
Output directorydist
Node.js version20
4
Click Deploy
Your site is live on a *.pages.dev subdomain. Add a custom domain in the Pages settings at any time.
5
(Optional) Add Cloudflare Web Analytics
In the Cloudflare dashboard → Analytics & Logs → Web Analytics → Add site → Copy the beacon token → Paste into analytics.cloudflareBeacon in site-config.ts.
Security headers (CSP, HSTS, X-Frame-Options, etc.) are pre-configured in wrangler.toml. They're applied automatically on Cloudflare Pages.

FAQ

How do I change my profile photo?

Update the identity.avatar URL in site-config.ts. You can use any public image URL, or add a file to public/ and reference it as "/my-photo.jpg".

How do I hide a link without deleting it?

Set enabled: false on any item. It will be invisible on the page but preserved in the code for later.

Can I add my own platform icon?

Yes — use platform: "custom" with icon: "lucide:star" (any Lucide or Simple Icons name) and color: "#your-hex".

How do I turn off the "Powered by Zutrabio" footer?

Set features.showPoweredBy: false in site-config.ts.

Can visitors change the theme?

Yes, via the floating theme switcher (palette icon). To prevent this, set theme.showThemeSwitcher: false.

Why does the promo banner not show after I dismissed it?

The dismiss state is stored in localStorage under the key zutrabio-promo-dismissed. Clear it in DevTools → Application → Local Storage to show the banner again during development.

How do I add a second language?

Add entries to seo.alternateLanguages in site-config.ts, e.g. [{ lang: "es", url: "https://es.yoursite.com" }]. This generates hreflang meta tags.
